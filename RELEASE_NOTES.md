# Mac Junk Scanner 0.1.0

Mac Junk Scanner 的首个公开版本现已发布。

这是一个面向 macOS 的原生中文清理工具，专注于帮助你更安全地识别和处理缓存、日志、开发产物以及疑似卸载残留文件。首版重点放在“看得清、选得明白、尽量避免误删”。

## 新增内容

- 原生 SwiftUI macOS 界面
- 常见垃圾目录扫描
- 已卸载应用残留识别
- 风险分级与逐项勾选
- 默认“移到废纸篓”而不是直接删除
- 废纸篓彻底清空入口
- 权限不足分类单独展示
- 首次启动权限引导

## 这一版适合谁

- 想清理缓存、日志、Xcode 产物的开发者
- 想检查 Mac 上残留文件占用空间的普通用户
- 希望在清理前先看原因、命中规则和风险等级的用户

## 使用建议

- 优先处理低风险项目，例如缓存、日志、DerivedData
- 模拟器数据、Xcode 归档、设备备份请谨慎处理
- 如果看到“需要授权”的分类，请先为应用开启“完全磁盘访问”

## 下载说明

- `MacJunkScanner.dmg`：推荐大多数用户下载
- `MacJunkScanner.app.zip`：适合手动解压使用

## 已知说明

- 当前 GitHub Releases 产物默认未做 Developer ID 签名和 notarization
- 首次运行时，macOS 可能要求右键应用后选择“打开”
- 部分系统目录的扫描结果取决于当前隐私权限
